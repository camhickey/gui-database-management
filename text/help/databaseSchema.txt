<p>The following are the columns and the corresponding data types of the database in which the class records are held. These columns are what should be used when defining parameters in a custom query. Not that they are not the exact same as the Excel column names in some cases; also note the "semester" and "year" columns at the end of the list. These two columns are not from the Excel file at all. This database is meant to run on PostgreSQL, so the datatypes should be taken from this documentation.</p>
